# The Spirit of Nuswantoro

A 3D adventure game developed using Unity and C# as part of a bachelor's thesis project.

## Overview

The Spirit of Nuswantoro is a third-person adventure game focused on exploration, mission-based progression, and combat mechanics. Players travel across multiple stages, complete objectives, and defeat spirit enemies using an ancestral bow.

## Features

- Third-person character controller
- Exploration-based gameplay
- Mission and objective system
- Stage progression system
- Auto-aim bow combat mechanic
- Enemy interaction and combat
- User interface (UI) implementation
- Audio integration
- Game state management

## Gameplay

Players explore different environments, complete objectives, and defeat spirit enemies to progress through each stage. The game encourages exploration while providing light combat and mission-based challenges.

## Technologies Used

- Unity
- C#
- Blender
- Git
